# 多人fps射击游戏的开发

## 人物方面需求：

1.人物的模型
2.人物的动作
3.人物动作的声音
4.需要第一人称视角进行操作
5.至少2种人物方便分队

## 地图方面需求：

1.场景的资产与搭建
2.一些可交互场景的设计

## 武器方面需求：

1.武器资产
2.武器的动画资产(包括第一人称换弹射击等）
3.多种武器的设计
4.武器的声音资产

## 游玩需要的要求：

- [ ] 1.游戏是第一人称视角

- [ ] 2.要有团队模式和个人模式（有爆破模式最好）

- [ ] 3.有开始大厅（csgo的开始登陆界面）和可创建房间选择模式和地图选择

- [ ] 4.（如果能实现）实现好友功能和可视化现有房间和加入的功能

- [ ] 5.游戏内可购买装备界面

- [ ] 6.游戏基本的一些ui如血量，武器显示团队比分，倒计时等

- [ ] 7.基本设置功能esc按键出现的交互菜单

- [ ] 8.按住tab出现的可以查看所有人战绩和网络延迟等的菜单

- [ ] 9.小地图功能

- [ ] 10.人物的死亡和相应模式的复活

- [ ] 11.枪械后坐力的实现

- [ ] 12.烟雾弹和闪光弹的实现

- [ ] 13.人物的静步蹲下跳跃

- [ ] 14.准星的缩放

- [ ] 15.击杀显示和反馈

- [ ] 16.死亡回放

- [ ] 17.死亡后复活过程中队友视角

- [ ] 18.对应模式的死亡是否掉落装备

- [ ] 19.经济系统击杀奖励

- [ ] 20.枪械的扔出和更换

- [ ] 21.近战匕首的实现

- [ ] 22.打击不同部位伤害的不同

- [ ] 23.护甲值的实现

- [ ] 24.复活后的无敌时间

- [ ] 25.游戏开始的准备阶段

- [ ] 26.可更改式键位

- [ ] 27.可调节式游戏声音

- [ ] 28.可设置的游戏准星

- [ ] 29.战斗结算界面
