# C++

## 1.被调用头文件在.h文件调用还是在.cpp中调用

```c++
//在不同类中调用其他类去创造变量需要知道这个变量的大小
//如:现在使用的类 Class_A,需要调用的类 Class_B
//需要在Class_A中调用Class_B
//还有Class_C和Class_D调用了Class_A

//故如果是非指针变量需要在Class_A.h文件中调用该类的头文件
//在Class_A.h调用头文件的影响导致Class_B在更改时进行编译会导致调用Class_A的Class_C和Class_D也进行编译增加了编译的速度
//而且C中指针都是32字节固定大小所以只要在Class_A.h中class Class_B声明一下可以避免上面的多次编译
//而在CLass_A.cpp中去调用Class_B.h即可

```

## 2.static成员函数

```c++
//没有this指针了
//不可以调用类的非静态成员
//静态成员函数属于这个类，不再仅仅属于具体的对象


//静态函数会被自动分配在一个一直使用的存储区，直到退出应用程序实例，避免了调用函数时压栈出栈，速度快很多。
```
